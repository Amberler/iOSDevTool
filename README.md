# iOSDevTool
基于Dart实现的iOS开发辅助工具，svn打tag，版本号自增，自动提交组件的podspec到仓库。

## V 1.0.2
1.指定版本号提交，无需校验提交记录。仅支持单个提交
2.已知bugs修复


## V 1.0.1
1.指定模块名提交，无需校验提交记录。仅支持单个提交
2.自动提交，支持多个组件名，逗号分隔，需要校验当天提交记录
